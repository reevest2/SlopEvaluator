# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-03-25

### Added

- Initial 6-project solution structure
- 14-dimension health scoring with weighted aggregation
- Mutation testing engine with 10 Roslyn-based strategies
- Orchestrator for scan -> improve -> verify pipeline
- CLI with 8 commands: scan, improve, gate, mutate, quality, compare, history, fix
- GitHub Actions CI with self-scan quality gate
